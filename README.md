# React Routing Concepts

This project demonstrates how to use React Router for navigation and routing in React applications.

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/mohan-parajuli/react-router.git
   cd react-router
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the application:
   ```
   npm start
   ```

## Features
- Dynamic routing
- Nested routes
- URL parameters
- 404 Not Found page
